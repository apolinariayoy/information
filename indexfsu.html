Формулы сокращенного умножения

  • Разность квадратов

    a2- b2 = (a - b)(a + b)

  • Квадрат суммы двух чисел

    (a + b)2 = a2 + 2ab + b2

  • Квадрат разности

    (a - b)2 = a2 - 2ab + b2

  • Сумма кубов

    a3 + b3 = (a + b)(a2 - ab + b2)

  • Разность кубов

    a3 - b3 = (a - b)(a3 + ab + b2)

  • Куб разности

    (a - b)3 = a3 - 3a2b + 3ab2 - b3

  • Куб суммы двух чисел

    (a + b)3 = a3 + 3a2b + 3ab2 + b3

    Пример

  • (x3 + 2)2(x3 + 2)2

  • Решение

  • можно возвести в квадрат результат произведения множителей:

  • (x3 + 2)2(x3 + 2)2 = ((x3 + 2)(x3 + 2))2

  • мы получили выражение, которое можно упростить по формуле разности квадратов:

  • ((x3 + 2)(x3 + 2))2 = (x6 + 4)2

  • Осталось возвести выражение в квадрат по формуле квадрата разности:

  • (x6 + 4)2 = x12 + 8x6 + 16
